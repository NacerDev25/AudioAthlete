# AudioAthlete 🎙️🏋️‍♂️

**AudioAthlete** is a smart, accessible, and modern workout timer designed for high-intensity interval training (HIIT). It focuses on **audio guidance**, allowing athletes to stay focused on their workout without needing to look at their screens.

## 🚀 Features

-   **Audio Guidance (Web Speech API):** The app speaks to you! It announces rounds, rest periods, and 10-second warnings.
-   **Automatic Round Calculation:** Simply enter your total workout time, work interval, and rest interval, and the app does the math for you.
-   **Fully Accessible (A11y):** Built with semantic HTML and ARIA labels, making it fully compatible with screen readers.
-   **Modern & Responsive UI:** A sleek dark-mode design that looks great and works perfectly on mobile, tablet, and desktop.
-   **No Hands Needed:** Once you start, the voice alerts keep you on track.

## 🛠️ Tech Stack

-   **HTML5:** Semantic structure and accessibility.
-   **CSS3:** Modern styling with CSS variables and responsive design.
-   **JavaScript (Vanilla):** Core logic and Web Speech API integration.

## 📖 How to Use

1.  Set your **Total Workout Time** in minutes.
2.  Set your **Work Interval** and **Rest Interval** in seconds.
3.  Click **Start Workout**.
4.  Follow the voice instructions and focus on your training!
